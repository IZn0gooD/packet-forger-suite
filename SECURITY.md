# Security Policy

## Reporting a Vulnerability

This is a personal research/hobby project maintained on a best-effort basis.

If you discover a security vulnerability, please report it privately via
[GitHub Security Advisories](../../security/advisories/new) rather than opening
a public issue.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs, PoC code, or configuration

There is no guaranteed response time, but reports will be reviewed and
addressed as time allows.

## Supported Versions

This project does not currently maintain multiple supported versions.
Only the latest commit on the default branch is considered current.
